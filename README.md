# Set

Set is game where you have to find sets of 3 cards that all either share no properties or all share a property.

### [Play it here](https://lorgan3.github.io/set/)

## Running

You need to have Node & Yarn installed. Simply run the following commands to start the game.

```
yarn
yarn dev
```

The server will start on [http://localhost:3000/](http://localhost:3000/)
